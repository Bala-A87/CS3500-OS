CS3500 Lab 8 - File Systems

CONTENTS OF SUBMISSION:

The submitted zip file will expand to a folder, 'CS20B012', containing the following:
    - this README,
    - the source code, disk-directory.cpp,
    - a Makefile,
    - a folder, 'Screenshots', containing screenshots of the output


RUNNING THE CODE:

The code can be compiled with the help of the Makefile, by running either 'make' or 'make all'.
Compilation will produce an executable named 'disk-directory'.
The executable can be run according to the following specifications:
    Usage: ./disk-directory <D>
    Where D is the disk size in KB
The generated executable can be removed by running 'make clean'.
